# Estudios Bíblicos - Antítesis

Este proyecto contiene una colección de estudios bíblicos que exploran diversos temas teológicos, con un enfoque en el análisis de diferentes perspectivas doctrinales.

## Estructura del Proyecto

```
antitesis/
├── css/
│   └── styles.css         # Estilos principales
├── js/
│   └── main.js            # Funcionalidad interactiva
├── slides/                # Diapositivas relacionadas (si aplica)
├── *.html                 # Archivos HTML de los estudios
└── README.md              # Este archivo
```

## Estudios Disponibles

1. **Antítesis del Clamor Divino** - Análisis sobre el clamor divino y su contraste con doctrinas modernas.
2. **Gracia vs Torá** - Exploración del equilibrio entre la gracia divina y la ley de Moisés.
3. **Shabbat y la Ley** - Estudio detallado sobre el día de reposo y su relevancia actual.
4. **El Matrimonio en la Torá** - Análisis de los principios bíblicos sobre el matrimonio según la Torá.
5. **Respuesta al Sermón del Monte** - Estudio detallado del Sermón del Monte.

## Cómo Usar

1. Abre el archivo `index.html` en tu navegador para ver todos los estudios disponibles.
2. Navega a través de los diferentes estudios haciendo clic en los enlaces.
3. Cada estudio es autónomo y puede ser compartido individualmente.

## Personalización

- **Estilos**: Modifica los estilos en `css/styles.css`
- **Funcionalidad**: Ajusta el comportamiento en `js/main.js`
- **Contenido**: Edita los archivos HTML individuales para actualizar el contenido de cada estudio

## Requisitos

- Navegador web moderno (Chrome, Firefox, Safari, Edge)
- Conexión a Internet (para fuentes de Google y recursos externos)
